# Data Engineering Pipeline Simulator

An interactive tool to simulate and visualize data engineering pipelines efficiently. This repository contains a Streamlit application for working with datasets, diagnosing issues, and simulating corrections in data pipelines.

## Features
- Process various datasets to simulate pipelines interactively.
- Detect and analyze corrupt data vs clean data.
- Streamlit-based GUI for simplicity and adaptability.
- Provides modular setup for customizing the environment.

## Getting Started

### Prerequisites
Install the necessary dependencies by running:
```bash
pip install -r requirements.txt
```

### Installation
1. Clone the repository locally:
   ```bash
   git clone https://github.com/GeeteshSingh/data-engineering-pipeline-simulator.git
   ```
2. Navigate into the project directory:
   ```bash
   cd data-engineering-pipeline-simulator
   ```
3. Follow the setup instructions:
   - Refer to the [Setup Guide](SETUP-GUIDE.md) for detailed installation steps.
   - Example of starting the Streamlit app:
     ```bash
     streamlit run streamlit_app.py
     ```
